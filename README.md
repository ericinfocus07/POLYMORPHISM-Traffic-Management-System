# POLYMORPHISM-Traffic-Management-System
A Python OOP project demonstrating polymorphism through a Smart Traffic Management System using inheritance and method overriding. 

A parent class named `TrafficDevice` defines a common method called `activate()`. Each child class overrides this method to perform a different task.

The project illustrates **run-time polymorphism (method overriding)**, where the same method call results in different behaviors depending on the object.

---

## Classes

### Parent Class
- TrafficDevice

### Child Classes
- TrafficLight
- SpeedCamera
- PedestrianSignal
- EmergencySiren

---

## Features

- Inheritance
- Method Overriding
- Run-time Polymorphism
- Dynamic Method Dispatch
- Easily extendable by adding new traffic devices without modifying existing code.
